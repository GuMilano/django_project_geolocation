# Yelp Data Analysis / Integration Project

Projeto desenvolvido para explorar, analisar ou consumir dados da plataforma **Yelp**, utilizando técnicas de programação, manipulação de dados e análise de informações relacionadas a estabelecimentos, avaliações e usuários.

O objetivo deste repositório é demonstrar a utilização de dados do **Yelp Dataset** ou da **API do Yelp** para fins de estudo, desenvolvimento de aplicações e análise de dados.

---

# 📊 Sobre o Yelp

O **Yelp** é uma plataforma amplamente utilizada para encontrar e avaliar negócios locais como restaurantes, cafés, hotéis e serviços.

A plataforma disponibiliza conjuntos de dados contendo:

- Informações sobre negócios
- Avaliações de usuários
- Dados de usuários
- Localização de estabelecimentos
- Classificação por estrelas
- Comentários e reviews

Esses dados são frequentemente utilizados para:

- Pesquisa acadêmica
- Projetos de ciência de dados
- Machine Learning
- Análise de mercado
- Desenvolvimento de aplicações baseadas em reviews

---

# 🚀 Objetivos do Projeto

Este projeto tem como principais objetivos:

- Demonstrar a utilização de dados do Yelp em aplicações reais
- Realizar análise de avaliações e negócios
- Explorar padrões de comportamento dos usuários
- Desenvolver scripts para coleta ou processamento de dados
- Criar visualizações e insights a partir das informações disponíveis
